# Bilder für den Adventskalender

Lege hier deine Bilder für die einzelnen Türchen ab.

## Benennung
- `tag1.jpg` - Bild für Tag 1
- `tag2.jpg` - Bild für Tag 2
- usw.

## Unterstützte Formate
- JPG/JPEG
- PNG
- GIF
- WebP

## Beispiel
Wenn du ein Bild für Tag 1 hast, speichere es als `tag1.jpg` in diesem Ordner und ändere in `script.js`:
```javascript
{
    day: 1,
    title: "Der Anfang",
    image: "images/tag1.jpg",
    content: `...`
}
```
